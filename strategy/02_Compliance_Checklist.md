# Compliance & QA Checklist

> Print this. Tape it next to your monitor. Use it on every video. The checks here are the difference between a channel that compounds and one that gets demonetized in month 4.

---

## Pre-publish checklist (every video, no exceptions)

### Content originality
- [ ] Each script has at least ONE original element: opinion, novel framing, original data, personal observation, or curated angle. (This is your defense against YouTube's "inauthentic content" policy.)
- [ ] Hook is not a verbatim copy of any of your last 10 videos
- [ ] Format is not identical to your last 5 videos in this niche
- [ ] Visual style varies from previous video (different b-roll pacing, different transitions, different cover frame style)

### Factual integrity
- [ ] Every specific number, date, name, or quote in the script has been verified against a primary source
- [ ] Sources are listed in the description (2–4 URLs minimum for fact-heavy videos)
- [ ] No claims that I personally cannot defend if challenged
- [ ] No medical, legal, or specific financial advice (educational framing only)

### AI disclosure
- [ ] If voiceover is synthetic and sounds realistic, the YouTube "altered or synthetic content" label is set
- [ ] Channel-level AI disclosure is present in the About section
- [ ] No real-person voice cloning (zero tolerance)
- [ ] No real-person likeness in generated images

### Copyright & licensing
- [ ] All stock footage from confirmed sources (Pexels / Pixabay / Mixkit / paid stock library)
- [ ] All music from platform-approved library (YouTube Audio Library / TikTok Commercial Music Library) OR confirmed royalty-free
- [ ] No clips of films, TV shows, sports, news broadcasts, or copyrighted content
- [ ] All AI-generated images: confirmed the model's commercial-use license permits this use

### Platform-specific format
- [ ] Aspect ratio: 1080×1920 (9:18 portrait)
- [ ] Length: 30–60s (or up to 90s with strong reason)
- [ ] Top 14% and bottom 18% of frame kept clear of essential text/visuals (platform UI sits there)
- [ ] Captions burned in, sized for mobile, high contrast
- [ ] Audio normalized to -14 LUFS, no clipping

### Metadata
- [ ] Title is specific, not clickbait, under platform character limits
- [ ] Description includes 1-line summary, lead-magnet/affiliate link, sources, AI disclosure note
- [ ] Hashtags: 3–5 highly relevant, not 30 generic ones
- [ ] Cover frame readable at thumbnail size
- [ ] CTA matches the niche's funnel stage (link in bio, save, comment for resource)

### Cross-platform variation
- [ ] Each platform variant has at least minor differences (intro card, opening seconds, audio bed)
- [ ] Posts spaced at least 60 minutes apart across platforms
- [ ] Different captions/descriptions per platform (don't paste identical metadata)

### Final human review
- [ ] I personally watched the final video end-to-end on a phone-sized screen
- [ ] I would not be embarrassed if a colleague in this niche saw this video
- [ ] No pronunciation errors in the voiceover
- [ ] No visual glitches, sync issues, or frame drops

---

## Weekly compliance review

Run every Sunday before planning next week.

- [ ] Reviewed any platform notifications, strikes, or warnings
- [ ] Checked YouTube Studio for any monetization status changes on individual videos
- [ ] Reviewed retention curves on this week's bottom-3 videos for any policy-related drop-off patterns
- [ ] Spot-checked 2 random videos from earlier this month: do they still meet the originality bar in hindsight?
- [ ] No automated systems posted while I was away from the dashboard for >24 hours

---

## Monthly compliance review

- [ ] Read the latest YouTube channel monetization policies page (https://support.google.com/youtube/answer/1311392) to check for updates
- [ ] Read the latest TikTok community guidelines updates
- [ ] Confirm all affiliate disclosures are still in place on every video's description
- [ ] Verify lead magnet / landing page disclaimers are current
- [ ] Audit last 30 days of videos: would an outside reviewer call any of them "templated" or "easily replicable at scale"? If yes, course-correct.
- [ ] Backup of this Project folder to cloud or external drive

---

## Red flags that should trigger a full pause

If ANY of these happen, stop publishing for at least 48 hours and investigate:

- Unexpected drop in monetization status on any platform
- Strike or community guideline warning on any video
- Sustained 30%+ drop in average retention across 5+ consecutive videos
- Multiple videos auto-flagged for AI content without your disclosure
- Affiliate program asking for compliance review

Pausing is cheap. Recovery from a channel-level demonetization is expensive (or impossible).

---

## Things I will never do, even if it would be efficient

- ❌ Clone the voice of a real person (celebrity, politician, anyone)
- ❌ Use copyrighted clips even briefly under "fair use" reasoning
- ❌ Auto-publish without seeing the final file myself
- ❌ Buy followers, views, likes, or engagement
- ❌ Post identical files to multiple platforms within minutes
- ❌ Run multiple channels in the same niche from the same account family
- ❌ Skip the fact-check stage to ship faster
- ❌ Leave automated publishing running while I'm offline for >24 hours
